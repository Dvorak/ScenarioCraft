# Imported Road Assets

This directory is reserved for future user-provided or public OpenDRIVE assets.

Imported roads are not trusted canonical ScenarioCraft roads by default. Each
imported road should be accompanied by provenance, license information, and a
road card describing topology/capability and known limitations before it is
considered for scenario-family binding.

Canonical ScenarioCraft-owned roads live under `assets/roads/canonical/` and
are generated from `scenariocraft/core/roads/`.
